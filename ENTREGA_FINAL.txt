💡 INSIGHTS PRINCIPAIS PARA EQUIPE MÉDICA
================================================================================

FATORES DE RISCO MAIS IMPORTANTES:
1. BMI ELEVADO
   - Preditor mais forte (30% de importância)
   - Forte correlação com todas as classes de obesidade
   - Ação: Calcular e monitorar em todos os pacientes

2. FALTA DE ATIVIDADE FÍSICA
   - FAF=0 aumenta risco significativamente
   - Ação: Recomendar mínimo 1x/semana

3. CONSUMO ALTO DE ALIMENTOS CALÓRICOS
   - FAVC=yes mostra forte associação
   - Ação: Avaliar padrão alimentar

4. HISTÓRICO FAMILIAR POSITIVO
   - Aumenta predisposição em ~2x
   - Ação: Investigar e monitorar

FATORES PROTETORES:
1. Atividade física regular (3-5x/semana)
2. Consumo adequado de vegetais (FCVC=3)
3. Alto consumo de água (CH2O>2L/dia)
4. Monitoramento de calorias

PROTOCOLO RECOMENDADO:
□ Calcular BMI de todos os pacientes
□ Avaliar frequência de atividade física
□ Revisar padrão alimentar (frequência de calorias)
□ Verificar consumo de água diário
□ Investigar histórico familiar
□ Classificar usando modelo
□ Implementar plano de intervenção personalizado

📝 COMO USAR O SISTEMA
================================================================================

INSTALAÇÃO LOCAL:
1. Clone o repositório GitHub
2. Instale dependências: pip install -r requirements.txt
3. Execute: streamlit run streamlit_app.py
4. Acesse: http://localhost:8501

USANDO A APLICAÇÃO:
1. Abra a aba "Previsão"
2. Preencha os dados do paciente
3. Clique em "Fazer Previsão"
4. Visualize resultado e recomendações

ACESSAR ANÁLISES:
- Aba "Análise de Dados": Insights gerais
- Aba "Sobre o Modelo": Performance e comparações
- Aba "Dicionário": Referência de variáveis
